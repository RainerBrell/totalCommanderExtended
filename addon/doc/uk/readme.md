# Total Commander Enhanced

**Автор:** Rainer Brell

*Цей додаток підтримується на пам'ять про Ральфа Кефферпютца.*

---

Total Commander Enhanced покращує роботу NVDA зі списками файлів у Total
Commander. Навігація по файлах і папках стає зручнішою — особливо для
користувачів брайлівського дисплея.

## Що робить цей додаток

**Брайлівський дисплей**

- На початку кожного рядка відображається буква, що вказує, в якій області
  ви знаходитесь (наприклад, ліва або права файлова область).
- Буква **мала**, коли елемент не виділено,
  і **ВЕЛИКА**, коли він виділений.
- Наприкінці рядка відображається позиція у списку, наприклад `3/42`.

**Синтез мовлення**

- NVDA повідомляє позицію у списку у природній формі,
  наприклад *«3 із 42»*.
- Мовленнєвий і брайлівський виводи є незалежними — префіксна буква
  відображається лише на брайлівському дисплеї та ніколи не промовляється.

**Загальне**

- Роздільники стовпців у назвах файлів відображаються як пробіли,
  що робить рядки зручнішими для читання.
