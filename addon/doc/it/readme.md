# Total Commander Enhanced

**Autore:** Rainer Brell

*Questo componente aggiuntivo è mantenuto in memoria di Ralf Kefferpuetz.*

---

Total Commander Enhanced migliora il funzionamento di NVDA con gli elenchi
di file in Total Commander. Navigare tra file e cartelle diventa più semplice,
in particolare per chi utilizza un display braille.

## Cosa fa questo componente aggiuntivo

**Display braille**

- All'inizio di ogni riga compare una lettera che indica in quale area ci si trova
  (ad esempio l'area file sinistra o destra).
- La lettera è **minuscola** quando una voce non è selezionata,
  e **MAIUSCOLA** quando è selezionata.
- Alla fine della riga è indicata la posizione nell'elenco, ad esempio `3/42`.

**Sintesi vocale**

- NVDA annuncia la posizione nell'elenco in modo naturale,
  ad esempio *"3 di 42"*.
- L'uscita vocale e quella braille sono indipendenti — la lettera iniziale
  appare solo sul display braille e non viene letta dalla sintesi vocale.

**Generale**

- I separatori di colonna nei nomi dei file vengono visualizzati come spazi,
  rendendo le righe più facili da leggere.

## Installazione

Il componente aggiuntivo può essere scaricato direttamente dall'Add-on Store
di NVDA e installato da lì. Non sono necessarie ulteriori impostazioni.
