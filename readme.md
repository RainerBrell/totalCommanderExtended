# Total Commander extended

**Author:** Rainer Brell

*This add-on is maintained in memory of Ralf Kefferpuetz.*

---

Total Commander extended  improves how NVDA works with the file lists in
Total Commander. It makes navigating files and folders easier — especially
for braille display users.

## What this add-on does

**Braille display**

- A letter at the beginning of each line shows which panel you are in
  (for example the left or right file panel).
- The letter is **lowercase** when an item is not selected,
  and **UPPERCASE** when it is selected.
- At the end of each line you see your position in the list,
  for example `3/42`.

**Speech**

- NVDA announces your position in the list naturally,
  for example *"3 of 42"*.
- Speech and braille output are independent of each other —
  the prefix letter is shown on the braille display only and is never spoken.

**General**

- Column separators in file names are displayed as spaces,
  so lines are easier to read.

## Installation

Download the add-on from the NVDA Add-on Store and install it from there.
No further configuration is needed.
